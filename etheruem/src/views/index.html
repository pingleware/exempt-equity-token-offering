Exempt Equity Smart Contract Integration

Steps:

  1. Install truffle, gnache-cli, and live-server globally. [npm i -g truffle gnache-cli live-server]
  2. Open a command/terminal session and start gnache-cli [gnache-cli]
  3. Edit truffle-config.js and configure the development setting for networks [host=127.0.0.1, port=8545, network_id=*]
  4. Open a second command/terminal session and start truffle [truffle migrate -network development]
  5. After contracts have been deployed, change to src path in the truffle session and start live-server [cd src && live-server]
  6. Copy the contract address from the ganache-cli or truffle session and paste in the field in the browser session
  7. Choose the contrat to interact
